Allow your customers to take an appointment via PushRDV.com

== Description ==

FR :
La plateforme PushRDV (www.pushrdv.com) vous permet de gérer des agendas et offre la possibilité à vos clients de réserver des rendez-vous en ligne
sur vos agendas. De plus, vous avez la possibilité de proposer des stages, des cours ou bien des évenements auxquels vos clients pourront s'inscrire.
Ce plugin a pour but l'intégration des stages et de la réservation des rendez-vous directement sur votre Site Wordpress.

EN :
PushRDV (www.pushrdv.com) is an online Scheduling application that allow your customers to set appointments in your calendar.
As a company you can also offer courses, stages, events and allow your customers to subscribe to these events (online payment is available).
This plugin allow you to integrate your stages and allow your customers to take their appointment directly on your Wordpress Site.


== Installation ==
FR:
1. Pour utiliser ce plugin vous devez posséder un compte PushRDV. (www.pushrdv.com)
2. Téléchargez le plugin.
3. Uploadez le dans le répertoire wp-content/plugins de votre site.
4. Activez le plugin.
5. Rendez-vous sur la page PushRDV du menu d'administration.
6. Utilisez votre ID PushRDV et votre Private Key pour vous authentifier. Vous trouverez ces informations dans le menu "Paramètres" -> "Configuration" -> "Préférences Générales" de PushRDV (www.pushrdv.com)
7. Vous pouvez maintenant vous servir du plugin.

EN:
1. You need a PushRDV account to use this plugin. (www.pushrdv.com)
2. Download this plugin.
3. Upload this plugin to the wp-content/plugins directory of your website.
4. Activate the plugin.
5. Go to the PushRDV page in the Admin Menu.
6. Use your PushRDV ID and Private Key to authenticate. These informations are available in the "Paramètres" tab of the PushRDV (www.pushrdv.com) menu under "Configuration" -> "Préférences Générales"
7. You can now use the plugin.

== Frequently Asked Questions ==

= How do I display my events ? =
Just use the button available (the one with the PushRDV logo) in the Wordpress Editor to add the shortcode.

= How do I use the shortcode ? =
You can use the WP Editor button or directly add these shortcode in the content : [agencyStages agency_shortname="<AGENCY_SHORTNAME>" /] where <AGENCY_SHORTNAME> is the shortname of one of your PushRDV agencies.

= Comment afficher mes stages ? =
Il vous suffit d'utiliser le boutton (avec le logo PushRDV) présent dans l'editeur de contenu Wordpress afin d'ajouter le shortcode.

= Comment utiliser le shortcode ? =
Vous pouvez vous servir du bouton présent dans l'éditeur Wordpress ou bien ajouter directement le shortcode suivant dans le contenu :
[agencyStages agency_shortname=" <AGENCY_SHORTNAME> " /]où <AGENCY_SHORTNAME> est à remplacé par le shortname d'une de vos agences PushRDV.
